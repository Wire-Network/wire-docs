## Description

Retrieves various items and information from the blockchain

## Subcommands

- [info](./info.md) - Get current blockchain information
- [finalizer_info](./finalizer-info.md) - Get current finalizer information
- [transaction-status](./transaction-status.md) - Get transaction status information
- [consensus_parameters](./consensus-parameters.md) - Get current blockchain consensus parameters
- [block](./block.md) - Retrieve a full block from the blockchain
- [account](./account.md) - Retrieve an account from the blockchain
- [code](./code.md) - Retrieve the code and ABI for an account
- [abi](./abi.md) - Retrieve the ABI for an account
- [table](./table.md) - Retrieve the contents of a database table
- [scope](./scope.md) - Retrieve a list of scopes and tables owned by a contract
- [currency balance](./currency-balance.md) - Retrieve currency balance info
- [currency stats](./currency-stats.md) - Retrieve information related to standard currencies
- [transaction_trace](./transaction_trace.md) - Retrieve a transaction from trace logs
- [block_trace](./block_trace.md) - Retrieve a block from trace logs
- [schedule](./schedule.md) - Retrieve the producer schedule
- [transaction_id](./deprecated/transaction_id.md) - Get transaction id given transaction object
- [supported_protocol_features](./supported-protocol-features.md) - Get supported protocol features
- [accounts](./deprecated/accounts.md) - Retrieve accounts associated with a public key (deprecated)
