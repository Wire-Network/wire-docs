## Command

```sh
clio version client [OPTIONS]
```

## Description

Retrieve basic version information of the client

## Synopsis

```sh
clio version client
  [-h | --help]                     # Print this help message and exit
  [--help-all]                      # Show all help
```

## Examples

The following examples demonstrate how to use the `clio version client` command:

### Get client version

```sh
clio version client
```

**Output:**

```console
v5.1.0
```
